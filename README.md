# Right Agent Group — Official Website

A premium, fully responsive single-page application for **Right Agent Group** — Hyderabad's trusted partner for Loans, Insurance, and Real Estate.

## 🌐 Live Site
**[rightagentgroup.github.io/rightagentgroup](https://whitewalker-dev.github.io/rightagentgroup)**

## ✨ Features
- 🌌 Premium dark glassmorphism UI with animated cosmic orbs
- 🖼️ Hero image slider with live API integration
- 📊 Animated stats counter
- 🏠 Full SPA routing — Home, About, Loans, Real Estate, Insurance, detail pages
- 📝 Lead capture forms with OTP verification
- 🤝 Partner bank logos ticker (HDFC, ICICI, SBI, etc.)
- ⭐ Customer reviews slider
- ❓ FAQ accordion
- 📱 Fully responsive mobile design
- 💬 WhatsApp floating chat button
- 🔔 Toast notification system

## 🛠️ Tech Stack
- **HTML5** — Semantic structure
- **Vanilla CSS** — ~2500 lines, glassmorphism, animations
- **Vanilla JS** — SPA router, live REST API calls, OTP flow
